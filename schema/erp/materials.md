**Schema**
| Header          | Description                                                         |
| --------------- | ------------------------------------------------------------------- |
| `material_id`   | Unique identifier (e.g., `RM0001`, `FG0001`)                        |
| `material_name` | Human-readable name (e.g., `PET Resin Grade A`, `500ml PET Bottle`) |
| `material_type` | `RAW_MATERIAL` or `FINISHED_GOOD`                                   |


**Example**
| material_id | material_name     | material_type |
| ----------- | ----------------- | ------------- |
| RM0001      | PET Resin Grade A | RAW_MATERIAL  |
| RM0002      | HDPE Resin        | RAW_MATERIAL  |
| RM0003      | Black Masterbatch | RAW_MATERIAL  |
| FG0001      | 500ml PET Bottle  | FINISHED_GOOD |
| FG0002      | 1L PET Bottle     | FINISHED_GOOD |
