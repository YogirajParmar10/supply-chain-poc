| Header                   | Description                                                                      |
| ------------------------ | -------------------------------------------------------------------------------- |
| `purchase_order_id`      | Unique identifier for the purchase order                                         |
| `order_date`             | Date the purchase order was created                                              |
| `supplier_id`            | Supplier receiving the purchase order                                            |
| `material_id`            | Raw material being ordered                                                       |
| `quantity`               | Quantity ordered                                                                 |
| `expected_delivery_date` | Expected delivery date                                                           |
| `status`                 | Current purchase order status (`CREATED`, `CONFIRMED`, `DELIVERED`, `CANCELLED`) |
