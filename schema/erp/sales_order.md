| Header                    | Description                                                                              |
| ------------------------- | ---------------------------------------------------------------------------------------- |
| `sales_order_id`          | Unique identifier for the sales order                                                    |
| `order_date`              | Date the sales order was created                                                         |
| `customer_id`             | Customer placing the order                                                               |
| `material_id`             | Finished good being ordered                                                              |
| `quantity`                | Quantity ordered                                                                         |
| `requested_delivery_date` | Requested delivery date from the customer                                                |
| `status`                  | Current sales order status (`CREATED`, `CONFIRMED`, `SHIPPED`, `DELIVERED`, `CANCELLED`) |
